### 👨‍💻 About Me

Backend engineer from Campinas, Brazil — focused on **architecture that scales and code that communicates**.

I design systems that are:

- Readable six months later  
- Testable without a running server  
- Explicit about state transitions  
- Resilient by design — not by luck  

Background in the MERN ecosystem, now building layered, domain-driven backends with strict TypeScript and clear architectural boundaries.

---

### 🧠 Engineering Philosophy

- Domain first, framework second  
- Boundaries before features  
- Business rules must live in one place  
- Errors should be semantic, predictable, and intentional  
- Systems should evolve without breaking core invariants
